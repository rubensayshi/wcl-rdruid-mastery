Resto Druid Analyzer

Hey fellow trees!

This tool works entirely in your browser without using a server backend to store or process things.
Because of this you need to get a warcraftlogs.com API key yourself, you can find this at the bottom of your accounts page; here

Everything will be stored inside your browser, so the API key is never leaked to the server or anything like that, don't worry ;)

This is the first, rather 'raw' version of this tool,
Keep in mind it could contain bugs! ...
